Plugin that displays a form to signup e-mails to selectecd list in Mailchimp

== Description ==
This plugin displays a simple form so that your users can subscribe to a selected list from Mailchimp.
U need a API Key from Mailchimp for this plugin to work

== Installation ==

e.g.

1. Upload folder `super-mailchimp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [mailchimp]
